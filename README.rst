fs.webdavfs
===========

``fs.webdavfs`` is a WebDAV driver for PyFileSystem2.


Supported Python versions
-------------------------

- Python 2.7
- Python 3.4
- Python 3.5
- Python 3.6

Repository
----------

- https://github.com/PyFilesystem/webdavfs

Issue tracker
-------------
